九余定理

九余定理是一个和数学有关的知识，可以了解一下

首先要介绍九余数的概念：一个数的各位数字之和相加后得到的<10的数字称为这个数（如果相加结果大于9，则继续各位相加）

然后是九余定理的定义：一个数 x，他的每个位上的数加起来为 y，九余数为（y - 1）再对 9 求余然后加1，用 C 语言表示就是 ( y - 1 ) % 9 + 1

emmm，就是说一个数，如果我要求它的九余数，怎么求呢，这样求？

代码一

#include <stdio.h>

int main(void)
{
    int s, t;
    int n;
    while ((scanf("%d", &n) != EOF) && (n != 0))
    {
        s = 0;
        for (;;)
        {
            for (;;)
            {
                s = s + n % 10;
                n = n / 10;
                if (n == 0)
                    break;
            }
            if (s < 10)
                break;
            n = s;
            s = 0;
        }
        printf("%d\n", s);
    }
    return 0;
}

这是我第一次做的时候的代码，发现在处理较大的数时行不通，所以就换一种，再看看这种？

代码二

#include<stdio.h>

int main(void)
{
    char x[10000];
    int i;
    while(scanf("%s", &x)&&(x[0] != '0'))
    {
        int t=0;
        for(i = 0; x[i]; ++i)
            t = t + x[i] - '0';
        t = t % 9;
        if(!t)
        {
            t = 9;
            if(x[0] == '0')                             //这里是特判 0 的情况
                t = 0;
        }
        printf("%d\n", t);
    }
    return 0;
}

就诞生了这个代码，没办法，因为处理较大的数的时候只好用字符来存储，但是还是不够简洁，下面就是九余定理

代码三

#include<stdio.h>
#include<string.h>

int main(void)
{
    char n[100000];
    int i;
    while((scanf("%s",n)!=EOF)&&n[0]!='0')
    {
        int s = 0;
        for(i = 0; i < strlen(n); i++)
            s = s + n[i] - '0';
        printf("%d\n", (s - 1) % 9 + 1);
    }
    return 0;
}
